# Log de Correções de Manuscritos

Cada correção feita pelo usuário é registrada aqui. Toda correção entra,
sem threshold (vide política em `_index.md`).

Formato:

```
## YYYY-MM-DD — <Cliente> (<Captador>)

### Pag <N> — Banco <X>
- Eu li: `<valor errado>`
- Correto: `<valor real>`
- Origem da correção: <usuário diz / cruzamento HISCON / etc>
- Observação: <opcional, padrão observado>
- Crop: `exemplos/<id>.png` (se anexado)
```
